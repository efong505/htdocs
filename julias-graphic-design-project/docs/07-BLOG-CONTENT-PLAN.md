## Blog Strategy for 2026

### Content Pillars
1. **Design Education** — Teach business owners about design principles (positions Julia as expert)
2. **Portfolio Case Studies** — Show real before/after work with the story behind it
3. **Industry Insights** — Trends, tools, and what's changing in graphic design
4. **Behind the Scenes** — Julia's artistic process, hand-drawn to digital workflow

### Posting Frequency
- **Goal:** 2 posts per month to start
- **Minimum:** 1 post per month
- Each post should be 600-1,000 words with images

## Blog Post Template

Every new blog post should follow this structure:

```
1. Hook (1-2 sentences that grab attention)
2. The problem or question
3. The answer/content (with images, examples, before/after)
4. Key takeaway
5. CTA: Free guide download OR consultation link OR shop link
6. Author bio snippet
```
